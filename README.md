# APIStudies

This Project is a study on how to implemented API funactions to Unreal engine 5. 
The tools used in this Project: Nodejs, Rest Api, UE5, Va Rest and Widgets.

<img width="1280" height="720" alt="image" src="https://github.com/user-attachments/assets/3ba3fdfc-b1c1-4427-abb3-16ebbcbf541f" />

A local server was created using Node.js for a todo list. With the help of the VaRest plugin for Unreal Engine, its API was integrated into the engine. Blueprints and widgets were then implemented to display the todo list in-game, with the ability to add new todos and modify existing ones.
