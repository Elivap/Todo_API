# APIStudies

This Project is a study on how to implemented API funactions to Unreal engine 5. 
The tools used in this Project: nodejs, rest api, ue5, va Rest and Widgets.

<img width="1280" height="720" alt="image" src="https://github.com/user-attachments/assets/3ba3fdfc-b1c1-4427-abb3-16ebbcbf541f" />

a local server is made trough nodejs with a code for a todo list and with the help of va Rest plugin for the Unreal Engine it implementet its api to the engine and made a code with blueprints and widgets to show the todo list ingame with the ability to add new todos to the list and changing already existing ones.
